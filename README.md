# 🎙️ AI Mock Interview

Practice job interviews with AI-generated questions and receive instant feedback to improve your technical knowledge, communication skills, and interview confidence.

---

## ✨ Features

- 🤖 AI-generated interview questions
- 💼 Role-based interview preparation
- 🧠 Technical & behavioral interview modes
- 📊 AI-powered feedback and evaluation
- 🔐 Secure user authentication
- 📝 Interview history tracking
- 📱 Fully responsive design

## 🚀 Getting Started

### Prerequisites

- Node.js
- npm

### Installation

```bash
git clone https://github.com/rutujbarsagade/Ai-Mock-Interview.git

cd Ai-Mock-Interview

npm install
```

### Environment Variables

Create a `.env.local` file:

```env
GOOGLE_GENERATIVE_AI_API_KEY=

NEXT_PUBLIC_FIREBASE_API_KEY=
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=
NEXT_PUBLIC_FIREBASE_PROJECT_ID=
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=
NEXT_PUBLIC_FIREBASE_APP_ID=
```

### Run the Application

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
